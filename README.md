# Zepensia 💎

[![Flutter](https://img.shields.io/badge/Flutter-02569B?style=for-the-badge&logo=flutter&logoColor=white)](https://flutter.dev)
[![Dart](https://img.shields.io/badge/Dart-0175C2?style=for-the-badge&logo=dart&logoColor=white)](https://dart.dev)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=for-the-badge)](https://opensource.org/licenses/MIT)

**Zepensia** is a sophisticated personal financial intelligence tool designed for high-performance tracking and secure data management. It provides a refined interface for monitoring your daily transactions with a focus on privacy and user experience.

---

## ✨ Key Features

- **🚀 Dynamic Dashboard**: Get a comprehensive overview of your financial status with real-time balance tracking and intuitive summary cards.
- **📑 Detailed History**: Navigate through your financial past with a beautifully structured transaction history.
- **📥 Smart Entry**: Quickly add income and expenses with an intelligent category bento-grid system.
- **🌍 Global Ready**: Support for multiple reporting currencies to track your wealth across borders.
- **🌗 Adaptive UI**: Experience a professional aesthetic in both Light and Dark modes, tailored for maximum readability.
- **🛡️ Data Sovereignty**: Complete control over your data with local SQLite storage and easy JSON export/import capabilities.
- **📅 Cycle Control**: Customize your accounting start day to align with your personal or business financial cycles.

---

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev)
- **Language**: [Dart](https://dart.dev)
- **Database**: [SQLite](https://www.sqlite.org/index.html) (via `sqflite`)
- **Architecture**: Clean, feature-based modular structure
- **State Management**: ValueListenableBuilder for efficient updates
- **Theming**: Custom Material 3 color schemes with professional typography

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (>= 3.2.0)
- Android Studio / VS Code with Flutter extension
- An Android or iOS device/emulator

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Dev-furqan-01/The-Ledger.git
   ```
2. Navigate to the project directory:
   ```bash
   cd zepensia
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the application:
   ```bash
   flutter run
   ```

---

## 📦 Data Management

Zepensia stores all data locally on your device for maximum privacy.
- **Database Path (Android)**: `/Documents/Zepensia/zepensia.db`
- **Exports**: Data can be exported as encrypted-ready JSON files for manual backup or migration.
